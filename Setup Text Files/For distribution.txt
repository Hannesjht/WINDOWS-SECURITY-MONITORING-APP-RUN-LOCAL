Creating a Windows .EXE from your Java Application


Method 1: Launch4j (Recommended - Free)

Launch4j wraps your JAR into an .exe with an embedded JRE option.

1. Download Launch4j: http://launch4j.sourceforge.net/


2. Create a new configuration:

   - Output file: `ai-security-monitor.exe`
   
   - Jar: `ai-security-2.0.0.jar`
   
   - Icon: Optional - you can create/use an .ico file
   
   - JRE: Set min/max version to `1.8.0`
   
   
3. Configure JRE options (optional):
   ```
   -Xmx512m
   
   -Dfile.encoding=UTF-8
   ```
   
   
4. Build the .exe


Method 2: jpackage (Built into JDK 14+)

This creates a native installer with embedded JRE.

bash


# Create a directory with your JAR

Creating a Simple .EXE with PowerShell (Quick method):

Create `create-exe.ps1`:


# Create a simple PowerShell wrapper .exe
$exeContent = @'
using System;
using System.Diagnostics;
using System.IO;

namespace AISecurityMonitor
{
    class Program
    {
        static void Main(string[] args)
        {
            string jarPath = "ai-security-2.0.0.jar";
            string javaPath = "java.exe";
            
            // Check for Java in JRE folder first
            if (File.Exists("jre\\bin\\java.exe"))
            {
                javaPath = "jre\\bin\\java.exe";
            }
            
            if (!File.Exists(jarPath))
            {
                Console.WriteLine("Error: Application JAR not found!");
                Console.WriteLine("Make sure ai-security-2.0.0.jar is in the same folder.");
                Console.ReadKey();
                return;
            }
            
            ProcessStartInfo psi = new ProcessStartInfo
            {
                FileName = javaPath,
                Arguments = $"-Xmx512m -jar \"{jarPath}\"",
                UseShellExecute = false,
                CreateNoWindow = false,
                RedirectStandardOutput = false
            };
            
            try
            {
                Process.Start(psi);
            }
            catch (Exception ex)
            {
                Console.WriteLine($"Error starting application: {ex.Message}");
                Console.WriteLine("Make sure Java 17+ is installed and in PATH.");
                Console.ReadKey();
            }
        }
    }
}
'@

# Save as C# file
$exeContent | Out-File -FilePath "Program.cs" -Encoding UTF8

# Compile to .exe (requires .NET Framework)
Add-Type -TypeDefinition $exeContent -Language CSharp -OutputAssembly "AI Security Monitor.exe" -OutputType ConsoleApplication

Write-Host "Created AI Security Monitor.exe" -ForegroundColor Green
```
