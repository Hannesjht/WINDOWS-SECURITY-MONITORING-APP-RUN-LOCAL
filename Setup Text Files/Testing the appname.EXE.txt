Troubleshooting Common Issues:

Issue 1: "Could not find the main class"

Solution: 

- Check main class name in config: `com.security.Main`

- Verify JAR contains the class: `jar tf ai-security-2.0.0.jar | findstr Main.class`


Issue 2: "Java not found" when JRE is present:

Solution:

- Ensure JRE folder structure: `jre\bin\java.exe`

- Check JRE version is 17+

- Test manually: `jre\bin\java.exe -version`


Issue 3: .exe crashes silently

Solution:

- Add debug options: `-Dlaunch4j.debug=true -Dlaunch4j.dump=true`

- Check Windows Event Viewer for crash logs

- Create a log file redirect in batch file


Issue 4: Icon not showing:

Solution:

- Ensure icon is .ico format (not .png renamed)

- Multiple sizes in icon (16x16, 32x32, 48x48, 256x256)

- Use icon editing software to create proper .ico
