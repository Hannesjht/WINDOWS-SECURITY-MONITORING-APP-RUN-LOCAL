# Build Instructions

1. Prerequisites:

   - JDK 17+
   - Inno Setup 6+ (for portable installer)


2. Build Application:
   bash
   
   mvn clean package
   # or
   ./gradlew build
   
   
System Requirements:

- Windows 7/8/10/11
- Java 17 or later
- 2GB RAM minimum
- Network access

Quick Start:

1. Double-click "AI Security Monitor.exe" or "launch.bat"
2. The application will start automatically
3. Click "Start Monitoring" in the Dashboard tab

Features:

- Real-time network monitoring
- AI-powered threat detection
- Port scanning capabilities
- Threat intelligence lookup
- Database logging
- Export functionality

Troubleshooting:

1. If the .exe doesn't run, try launch.bat
2. Ensure Java is installed: java -version
3. Check logs in the logs/ folder

Support:

For issues, check the logs folder for error details
